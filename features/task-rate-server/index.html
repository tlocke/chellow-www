Rate Server Importer

For most industry data we can use APIs, but some, such as DNO charging statements, have to be manually downloaded. We download them centrally to a repository which we call the Rate Server.

Clicking on the Automatic Importer: Rate Server link takes you to the page for importing the files from the rate server into Chellow. The importer runs once a day automatically by default, so you should rarely have to attend to it.

Rate Server Importer

For a new installation of Chellow it's a good idea to click the Import Now button to start an import. Click the browser refresh button to see the latest progress in the log. Make sure it completes successfully.
