The Open Source Web Application For Validating The Energy Bills Of Large UK Organisations

Chellow has been successfully checking bills since 2006. It is a browser-based system designed for organisations that spend over £1m / year on energy. Chellow works by creating expected bills, and then comparing them against the actual bills. Any differences are highlighted to the user so that they can can be taken up with the supplier. A strength of Chellow is the accuracy of its expected bills, derived from the supply contract and industry data feeds.

Chellow is Open Source software, so you'll never be locked in to a single vendor. You can download and run Chellow yourself, or contact us if you'd like a fully set-up hosted instance of Chellow.
