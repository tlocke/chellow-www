Contribute

Chellow is an Open Source project, and welcomes anyone who would like to contribute. Here's a non-exhaustive list of where help is needed:

  • This website. If you're keen on web design, visit the GitHub site for this website to start making improvements.
  • Write a blog post for this website.
  • Outreach. We need help in reaching people who may benefit from using Chellow, but don't know about it.
  • Using Chellow. It would be good to have more users.
  • Reporting bugs or requesting features using GitHub Issues.
  • Documentation.
  • Programming. You can clone Chellow and get started with fixing bugs or improving Chellow.
